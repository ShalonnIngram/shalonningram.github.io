Projects



  • Loan Data Project

    This project uses the AWS cloud infrastructure. Loads files into a S3 bucket, uses a Lambda function trigger to run a PySpark scipt over a EMR Cluster to transform data and outputs in parquet files, into a S3 bucket. A Glue Crawler crawls the S3 bucket for the schema to create a Glue Catalog. A Glue job is then ran to load data from Glue Data Catalog to Redshift Spectrum for querying. Redshift Spectrum is then connect to Tableau for Data Visualization and Reporting.
