Projects



  • Loan Data Project

    This project uses the AWS cloud infrastructure to load files into an S3 bucket, uses a Lambda function trigger to run a PySpark scipt over an EMR Cluster for data transformation and outputs parquet files, into an S3 bucket. A Glue Crawler crawls the S3 bucket for the data schema to create an Glue Catalog. A Glue job is then ran to load data from the Glue Data Catalog to Redshift Spectrum for querying. Redshift Spectrum is then connected to Tableau for data visualization and reporting.



  • Spotify Playlist Project

    In this project, data is extracted from Spotify's API, transformed with the Python Pandas library, loaded into Snowflake for data warehouse storage and connected to the BI tool Looker for data visualization and reporting.



  • Data Modeling Project

    In this project, I created a data model for a Postgres database based on a food ordering system. I also created a star schema data model design (fact & dimension tables) for a datawarehouse to allow OLAP data & efficient queries.
